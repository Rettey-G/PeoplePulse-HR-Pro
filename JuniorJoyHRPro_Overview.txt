# Junior Joy HR Pro: Happy Teams, Smarter HR
## Comprehensive System Overview

## INTRODUCTION
Junior Joy HR Pro is a modern, cloud-based human resources management system designed to streamline and enhance all aspects of HR operations. Built with a responsive design that works seamlessly on both desktop and mobile devices, it provides a comprehensive suite of tools for managing employees, tracking performance, processing payroll, handling leave requests, organizing training programs, and generating detailed analytics.

## SYSTEM ARCHITECTURE
- Frontend: React.js + Material UI, hosted on Netlify
- Backend: Node.js + Express, hosted on Render
- Database: MongoDB Atlas (cloud-based NoSQL database)
- Real-time Communication: Socket.io
- Version Control: GitHub
